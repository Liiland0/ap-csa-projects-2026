# AP CSA Projects 2026

This repository contains programming projects completed from my AP Computer Science A class during my junior year .

These projects focus on core computer science concepts including object-oriented programming, data structures, algorithms, and problem-solving.

## Projects Included

### Pathfinding (A* Algorithm)
2D grid pathfinding using the A* search algorithm.

This project focuses on:
- Heuristic-based search
- Grid traversal logic
- Shortest path optimization

---

### Doubly Linked Lists & Trees
Exploration of data structures including:
- Doubly linked lists
- Binary trees & binary search trees

Features:
- Node structured design
- Insertion and deletion functions
- Traversal and search features

---

### Conway’s Game of Life
Simulation of cells based on Conway’s rules.

Concepts used:
- 2D arrays / grids
- State updates per iteration
- Rule-based system design

---

### Recursion Practice (Towers of Hanoi)
Recursive solution to the Towers of Hanoi problem.

Concepts used:
- Recursive problem breakdown
- Stack-based thinking
- Algorithmic step tracing

---

### Sorting Algorithms Practice
Basic sorting algorithms including:
- Insertion Sort
- Selection Sort

Concepts used:
- Time / complexity basics
- Iterative algorithm design
- Array manipulation
